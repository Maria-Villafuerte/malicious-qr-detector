Objetivo

Desarrollar un sistema capaz de clasificar automáticamente imágenes de códigos QR en dos categorías: benignos (seguros) y maliciosos (potencialmente peligrosos). El sistema combina ingeniería de características tradicional con una red neuronal convolucional (CNN) para aprender patrones espaciales discriminativos que los métodos estadísticos globales no pueden capturar.

¿Qué es Random Forest?

Random Forest es un algoritmo de aprendizaje supervisado basado en conjuntos de árboles de decisión. Entrena múltiples árboles de forma independiente, cada uno sobre una muestra aleatoria del dataset y un subconjunto aleatorio de features. La predicción final se obtiene por votación mayoritaria entre todos los árboles.

Su fortaleza es que reduce el sobreajuste (overfitting) que tienen los árboles individuales, ya que la diversidad entre árboles cancela sus errores individuales.

¿Por qué usarlo aquí?

Se usa como modelo baseline para establecer un techo de rendimiento sobre las features tabulares extraídas manualmente (DCT, LBP, estadísticas de cuadrante). Si la CNN no supera este baseline, hay un problema.

✓ Entrenado (2,000 imágenes de muestra)

Configuración en este proyecto

  • n_estimators: 200 árboles
  • Datos: 2,000 imágenes (muestra balanceada)
  • Features entrada: 34 → reducido a 12 tras selección
  • Evaluación: Cross-Validation 5-fold
  • Escalado: StandardScaler aplicado antes del entrenamiento

Resultados obtenidos

  • 34 features: 72.90% ± 0.90% accuracy (CV)
  • 12 features seleccionadas: 73.25% ± 1.27% accuracy (CV)
  • Feature más importante: q_br_mean (cuadrante inferior derecho)
Limitación: ~73% de accuracy sugiere que las features tabulares no capturan suficiente información espacial. La CNN está diseñada para superar este techo.
Modelo: Red Neuronal Convolucional (CNN)
✓ Arquitectura implementada en cnn_model.py

¿Qué es una CNN?

Una Red Neuronal Convolucional (CNN) es un modelo de deep learning diseñado para procesar datos con estructura espacial como imágenes. En lugar de usar features calculadas manualmente, aprende automáticamente qué patrones visuales son discriminativos directamente desde los píxeles.

Sus capas de convolución aplican filtros que detectan bordes, texturas y formas a distintas escalas. Las capas más profundas combinan estas detecciones simples en representaciones complejas.

¿Por qué es superior a Random Forest aquí?

El análisis EDA demostró que las clases no son linealmente separables en el espacio de features tabulares. La CNN puede aprender la geometría interna de los QR codes — posición de módulos, patrones de detección de posición — que ninguna feature manual captura completamente.

Ventaja clave del diseño

La combinación de BatchNormalization en cada bloque convolucional y Dropout 0.4 en el clasificador está diseñada para evitar sobreajuste con 140K imágenes de entrenamiento.

Arquitectura CNN

Capas del Modelo

ConvBlock 1 Conv2d(1→32) + BN + ReLU + MaxPool → 64×64
ConvBlock 2 Conv2d(32→64) + BN + ReLU + MaxPool → 32×32
ConvBlock 3 Conv2d(64→128) + BN + ReLU + MaxPool → 16×16
ConvBlock 4 Conv2d(128→256) + BN + ReLU + MaxPool → 8×8
Conv Extra Conv2d(256→256) + BN + ReLU + AdaptiveAvgPool → 4×4
Classifier Flatten → FC(4096→512) → Dropout(0.4) → FC(512→128) → FC(128→2)
